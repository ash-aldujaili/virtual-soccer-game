# virtual-soccer-game
A virtual soccer game in a penalty-kick mode, implemented in `MATLAB`.
For demo, please refer to this [video](https://www.youtube.com/watch?v=1McrDAgRSlE).

# Related files:
In addition to the reposiotry code, you need the following:
1. `spfilt.m`: from Gonzalez et al.'s book on image processing. 
2. `bwlargestblob.m` from Mathworks.
3. `changeclass.m` from Mathworks.

Let me know if you spin something off this code ;) Enjoy
